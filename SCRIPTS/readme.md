# SCRIPTS

Questa cartella contiene una raccolta di **script utili per l'automazione di attività amministrative e operative**.

Gli script sono pensati per semplificare operazioni ripetitive, velocizzare procedure di gestione e supportare attività di manutenzione dei sistemi.

---

## Contenuti

All'interno della cartella è possibile trovare script relativi a:

- automazione di attività di sistema
- gestione file e directory
- configurazioni di rete
- operazioni di backup
- raccolta informazioni di sistema
- manutenzione e troubleshooting

---

## Tipologie di script

Gli script possono essere scritti in diversi linguaggi, ad esempio:

- Bash / Shell
- PowerShell
- Python
- Batch (Windows)

---

## Obiettivo

L'obiettivo di questa raccolta è:

- automatizzare attività ripetitive
- ridurre il tempo necessario per operazioni manuali
- mantenere uno storico di script utili
- facilitare la condivisione di strumenti tra ambienti diversi

---

## Note

Prima di eseguire uno script:

- verificare sempre il contenuto
- assicurarsi che sia compatibile con il proprio sistema
- testarlo in un ambiente controllato quando possibile
